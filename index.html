Introduction

Flow 1 Overview
Flow name Create brief + Edit brief
User role Teacher
Trigger (why does the user start this?) To create a new brief for students to work on and submit
End state (what does success look like?) To create a new brief and to be added to the “All Briefs” tab
# User Action System Response User's Goal Pain Point / Issue?
1 Clicked on “All Briefs” Opened page showing all created briefs To find where I can create a new brief It wasn’t clear if “All briefs” was just a place to view existing briefs or if there was more in there
2 Found and clicked “Create new brief” button Took me to a blank page to enter my brief title To begin creating the brief The button to create a new brief was very small and didn’t stand out immediately.
3 Entered brief title clicked “save” Took me to a page with some text boxes to enter details about my brief To Begin editing my brief It wasn’t clear how I could edit my brief or why I had to save first.
4 Entered a short description of the brief Entered the text in the text box To enter a short description of the brief I wasn’t sure how to save my progress
Flow 2 Overview
Flow name Manage requirements of brief
User role Teacher
Trigger (why does the user start this?) To add requirements for students to hit with their work
End state (what does success look like?) A list with checkboxes should show for students doing the brief
# User Action System Response User's Goal Pain Point / Issue?
1 Clicked “edit mode” Displayed a text box with some basic instructions above Find where to add requirements It wasn’t clear if this was how to add a checklist or if it was just a basic list
2 Followed instructions and added some requirements Added the text I typed into the text box Add a list with checkboxes to the brief It wasn’t clear if this was added the checkboxes or if I had to find a way to do that myself
3 Tried to save the changes No response To ensure my changes and requirements were saved to the brief The “brief saved” button is always greyed out so it’s confusing about how long ago it saved
4 Clicked “view mode” above the requirements Showed the checklist of requirements for students To maybe get a look at the brief as a whole as if a student were looking at it
5 Clicked on “All Briefs” and scrolled to see if my work was there Brought me to the briefs page To find my brief and check if the changes had saved Feels like an extra step when the “Save” button could just be used to make sure your work is saving
